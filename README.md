## Features

- **Real-time Rates**: Powered by `open.er-api` for broad currency support (including KZT, KGS, RUB).
- **Interactive Charts**: Visualize 30-day market trends using `chart.js`.
- **Operation History**: Automatically saves your recent calculations locally.
- **Dark Mode**: Fully supported dark theme for night-time usage.
- **Smart UX**: 
  - Custom, searchable currency selector with flags.
  - One-tap result copying.
  - Animated currency swapping.
  - Responsive glassmorphism design.
- **Tech Stack**: React 18, Tailwind CSS, Chart.js, Axios, Vite.

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
